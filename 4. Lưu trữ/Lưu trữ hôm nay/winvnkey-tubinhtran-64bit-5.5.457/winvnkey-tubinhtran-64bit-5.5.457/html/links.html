Vietnamese Computing Sites

 

  • WinVNKey:
    • Homepage at www.sourceforge.net.
    • Download latest keyboard drivers and Viscii/Unicode fonts.
    • Discussion forum available at either vn.net or VietUnicode.
    • Report WinVNKey bugs: submit report on line or send email.

    • Winvnkey-Developers Mailing List -- open to people interested in technical discussion of WinVNKey development only:
      • You can subscribe/unsubscribe on line.
      • You can subscribe, or unsubscribe , or get help by email.

  • Vietnamese Standardization Working Group (Viet-Std)
    The group that designed VISCII, MacVISCII, and worked closely with the Unicode Consortium on Vietnamese character encodings. This site contains reports on VISCII character set and VIQR.
     
  • The TriChlor Software Group
    The group that developed free software and fonts to support VISCII and Unicode on Windows, Macintosh, and Unix. You can visit this site and download software for Windows, Mac, Unix.
     
  • MacVNkey on sourceforge.net.
    • Download Macintosh keyboard driver and Vietnamese VISCII fonts for use on Mac OS classic.
    • Discussion forum at VietUnicode.
       
  • VietUnicode on sourceforge.net: If you ever want to know about Unicode and Vietnamese in Unicode, this is a must-visit site for you. It contains a wealth of information on Vietnamese Unicode characters, Unicode fonts, Unicode utilities/converters, and links to many Vietnamese computing sites, mailing lists, or forums.
    • Unicode Fonts for Vietnamese on Windows, Macintosh, as well as Microsoft Windows fonts that used to be available for free download at Microsoft website (e.g., Arial, Courier New, Times New Roman, Arial Unicode MS).
    • VietUnicode Support Forum provides many computing topics, including Fonts, WinVNKey and MacVNkey, UnicodeConverter, VietPad, VietIME, etc.
       
  • vn.net homepage: This site contains WinVNKey Homepage, WinVNKey forum, Vietnamese software and font downloads, music, geography, Viet Yellowpage, etc.
     
  • Microsoft website for Vietnamese.   Probably you should upgrade Microsoft fonts on your system if it is old.
     
  • Download free Arial Unicode MS font:
    • Download the exe file from a website. If the file is no longer available, you can search for download aruniupd.exe on google.com and may be able to find the file is available at some site.
    • Download the zip file from Phat Hoc Dao Uyen.

The compressed file is very large (almost 14 MB). The font file contains some 52,000 chars, including classic Vietnamese (Nôm), Chinese (Hán), Japanese, Korean, Pali-Sankrit (Pha.n), etc.
